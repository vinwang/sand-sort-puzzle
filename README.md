# Sand Sort Puzzle - 游戏开发模板

一个基于 **深海美学** 的液体分拣解谜游戏，支持 **Web 预览** 和 **微信小程序** 双端。

> 💡 本项目完全遵循 [Game Design Document v1.1]，包含动态关卡、Undo、玻璃试管视觉等特性。

## 🎮 在线试玩（本地）

1. 克隆本仓库或解压 ZIP
2. 进入 `sand-sort-puzzle-template` 目录
3. 启动本地服务器：
   ```bash
   # 使用 Python
   python3 -m http.server 8000

   # 或使用 Node.js
   npx serve -p 8000
